## Функциональность

### Обязательные требования (выполненые)
- Главная страница со списком всех стран в виде карточек (2 балла)  
  — Эмоджи флага, название, регион, столица, площадь, население, ссылка «Подробнее»  
- Поиск по названию страны (1 балл)  
- Фильтр по населению (мин/макс) (1 балл)  
- Фильтр по площади (мин/макс) (1 балл)  
- Фильтр по региону (элемент `<select>`) (1 балл)  
- Страница конкретной страны с подробной информацией (2 балла)  
- Бэкенд с двумя эндпоинтами:  
  — `GET /countries` — сокращённый список для главной страницы  
  — `GET /countries/:code` — полная информация о стране (обязательно)

### Дополнительные требования (выполненые все)
- Смена темы сайта: светлая / тёмная / системная (с сохранением выбора в localStorage)
- Возможность выбора режима отображения списка стран: карточки ↔ таблица 
- Изображение флага страны большого размера на странице подробностей 
- Отображение интерактивной карты, автоматически центрирующейся на стране

### Дополнительные улучшения 
- Адаптивная сетка карточек  
- Плавные анимации и hover-эффекты  
- Отображение дополнительной информации на странице страны: субрегион, валюты, официальные языки
